1) Посмотреть где я - pwd
2) Создать папку - mkdir folder_name
3) Перейти в папку - cd folder_name
4) Создать файл - cat >> file_name.txt
5) Выйти из редактора - комбинцаия ctrl+c
6) Просмотреть файл - cat file_name.txt
7) Редактировать файл - vim file_name.txt
8) Режим Insert - [i] (редактирование)
9) Выход из режима [i] - комбинация ctrl+c
10) Выход из vim - [:wq]
11) Поиск файла (имя целиком) - find . -name 'file_name.txt'
12) Поиск файла (часть имени) - find . -name '*file*'
13) Поиск ТОЛЬКО файлов - find . -type f - name '*file*'
14) Поиск ТОЛЬКО директорий - find . -type d - name '*file*'
15) Показать содержимое - ls -la (в т.ч. скрытые файлы)
16) Показать содержимое - ls (без учета скрытых файлов)
17) Создать файл touch file_name.txt
18) Удалить файл rm file_name.txt
19) Удалить НЕ пустую папку - rm -r folder_name
20) Вывести весь список комнад - history
21) Подняться на уровень выше в директории - cd ..
22) Поднять на 2 уровня выше в директории - cd ../../
23) Удалить ПУСТУЮ папку - rm dir folder_name
24) Копирование: cp ~/file_name.txt ~/folder_name/file_name.txt
25) Перемещение: mv ~/file_name.txt ~/folder_name/file_name.txt
26) Поиск данных в файле - grep искомая_фраза file_name.txt
27) Поиск точной фразы - grep -w искомая_фраза file_name.txt
28) Поиск строки в директории - grep -l -r искомая_строка .
29) Сложить результаты поиска в отдельный файл - grep -r искомая_фраза . > file_name.txt (имя файла, куда будут сложены резульаты поиска)
